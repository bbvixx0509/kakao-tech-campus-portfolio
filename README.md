# Portfolio

카카오테크캠퍼스 4기 지원을 위해 정리한 포트폴리오 저장소입니다.  
수업 과제와 개인 실습 결과물을 GitHub에서 바로 읽기 쉽게 다시 정리했습니다.

## What This Repository Shows

- C/C++ 콘솔 프로그램 구현 경험
- 자료구조와 그래프 탐색 알고리즘 실습
- JSP 기반 회원가입 및 로그인 웹 프로젝트
- R 기반 머신러닝 실습 코드
- Python 노트북 기반 데이터 분석 프로젝트

## Project Overview

### [jsp-web-portfolio](./jsp-web-portfolio)
회원가입, 로그인, 회원 관리 흐름을 구현한 JSP 웹 프로젝트입니다.  
페이지 역할이 분명해서 처음 보는 사람도 구조를 따라가기 쉽습니다.

### [c-development-projects](./c-development-projects)
전화번호부, 비디오 대여, 플래시 메모리 실습처럼 직접 동작하는 콘솔 프로그램 모음입니다.  
파일 저장, 메뉴 구성, 기능 분리를 어떻게 했는지 확인할 수 있습니다.

### [weather-subway-crowding-analysis](./weather-subway-crowding-analysis)
날씨와 지하철 혼잡도의 관계를 살펴본 데이터 분석 노트북입니다.  
주제를 정하고 데이터를 정리한 뒤 결과를 해석하는 흐름을 담았습니다.

### [machine-learning-portfolio](./machine-learning-portfolio)
분류, 회귀, 퍼셉트론, 신경망 기초 실습을 R 코드로 정리한 폴더입니다.  
각 코드가 어떤 값을 출력하는지와 그 의미를 함께 읽을 수 있게 정리했습니다.

### [c-algorithm-practice](./c-algorithm-practice)
연결 리스트, 큐, 힙, DFS, BFS를 직접 구현한 실습 코드입니다.  
기본 자료구조와 탐색 알고리즘을 코드로 옮기는 과정을 확인할 수 있습니다.

## Recommended Reading Order
