# 포트폴리오

카카오테크캠퍼스 4기 지원을 위해 정리한 포트폴리오 저장소입니다.  
대학교 4학년까지 직접 구현하고 분석한 프로젝트를, 심사자가 빠르게 핵심을 파악할 수 있도록 대표 프로젝트 중심으로 구성했습니다.

단순히 코드만 모아둔 저장소가 아니라, 각 프로젝트에서 어떤 문제를 다루었고 어떤 방식으로 구현했는지 바로 확인할 수 있도록 프로젝트별 README와 화면 예시를 함께 정리했습니다.

## 대표 프로젝트

### [JSP 기반 웹 애플리케이션](./JSP%20기반%20웹%20애플리케이션)
회원가입, 로그인, 회원 목록 조회, 삭제, 로그아웃까지 이어지는 기본 회원 관리 흐름을 JSP와 Bootstrap으로 직접 구현한 웹 프로젝트입니다.  
화면 흐름이 분명하고, 페이지별 역할 분리가 잘 보여서 가장 먼저 보기 좋은 대표 프로젝트입니다.

### [C 개발 프로젝트](./C%20개발%20프로젝트)
플래시 메모리 시뮬레이션, 전화번호 관리 프로그램, 비디오 대여 관리 시스템을 묶어 정리한 콘솔 프로그램 포트폴리오입니다.  
메뉴 설계, 파일 입출력, 구조체 기반 데이터 처리, 기능 분리 방식까지 확인할 수 있어 구현 역량을 보여주기 좋습니다.

### [날씨 기반 지하철 혼잡도 분석](./날씨%20기반%20지하철%20혼잡도%20분석)
날씨 정보와 지하철 혼잡도 사이의 관계를 분석하고 예측 성능을 검증한 데이터 분석 프로젝트입니다.  
데이터 전처리부터 모델 성능 확인, 결과 시각화까지 하나의 흐름으로 정리해 분석 프로젝트 대표작으로 배치했습니다.

## 프로젝트 구성

- [C 개발 프로젝트](./C%20개발%20프로젝트): 콘솔 환경에서 동작하는 C, C++ 프로그램을 모아둔 폴더입니다.
- [자료구조와 알고리즘 구현](./자료구조와%20알고리즘%20구현): 연결 리스트, 큐, 힙, DFS, BFS를 직접 구현한 코드 모음입니다.
- [JSP 기반 웹 애플리케이션](./JSP%20기반%20웹%20애플리케이션): 회원 관리 흐름을 중심으로 구성한 JSP 웹 프로젝트입니다.
- [개인 학습 관리 자동화 플러그인](./개인%20학습%20관리%20자동화%20플러그인): 학교 포털 일정 요약, 학습 자료 정리, 노션 발행 흐름을 구현한 보조 프로젝트입니다.
- [LLM 기반 가짜뉴스 판독기](./LLM%20기반%20가짜뉴스%20판독기): Upstage Solar, n8n, 텔레그램 봇을 연결해 기사와 영상의 사실 여부를 점검한 해커톤 프로젝트입니다.
- [머신러닝 포트폴리오](./머신러닝%20포트폴리오): 분류, 회귀, 신경망, 모델 성능 비교 코드를 주제별로 정리한 R 기반 머신러닝 포트폴리오입니다.
- [날씨 기반 지하철 혼잡도 분석](./날씨%20기반%20지하철%20혼잡도%20분석): Python과 Jupyter Notebook으로 진행한 예측 분석 프로젝트입니다.
