# 포트폴리오

카카오테크캠퍼스 4기 지원을 위해 정리한 포트폴리오 저장소입니다.  
C 기반 콘솔 프로그램, JSP 웹 프로젝트, 머신러닝 실습 코드, 데이터 분석 노트북을 한 저장소 안에서 보기 쉽게 정리했습니다.

이 저장소는 파일만 모아 둔 형태가 아니라, 각 폴더의 `README.md`에서 프로젝트 설명, 주요 파일 역할, 실행 결과 예시를 바로 확인할 수 있도록 구성했습니다.

## 대표 프로젝트

### [JSP 웹 프로젝트](./JSP%20웹%20프로젝트)
회원가입, 로그인, 회원 목록 관리, 로그아웃 흐름을 JSP로 구현한 웹 프로젝트입니다.  
페이지별 역할이 나뉘어 있어 웹 서비스의 기본 구조를 이해하기 쉽고, 실행 화면도 함께 정리해 두었습니다.

### [C 개발 프로젝트](./C%20개발%20프로젝트)
플래시 메모리 실습, 전화번호부, 비디오 대여처럼 콘솔 환경에서 직접 동작하는 프로그램을 모아 둔 폴더입니다.  
메뉴 구성, 파일 입출력, 구조체 활용, 기능 분리 방식이 드러나는 파일들을 중심으로 정리했습니다.

### [날씨와 지하철 혼잡도 분석](./날씨와%20지하철%20혼잡도%20분석)
날씨 정보와 지하철 혼잡도 사이의 관계를 분석한 데이터 프로젝트입니다.  
주제를 정한 뒤 데이터를 정리하고, 모델을 학습시키고, 결과를 해석하는 흐름을 노트북 하나로 확인할 수 있습니다.

## 전체 구성

- [C 개발 프로젝트](./C%20개발%20프로젝트): C와 C++로 작성한 콘솔 프로그램 모음입니다. 파일 저장, 메뉴 처리, 구조체 기반 데이터 관리 방식을 확인할 수 있습니다.
- [C 알고리즘 실습](./C%20알고리즘%20실습): 연결 리스트, 큐, 힙, DFS, BFS를 직접 구현한 코드 모음입니다. 자료구조와 탐색 알고리즘의 기본 구현 방식을 확인할 수 있습니다.
- [JSP 웹 프로젝트](./JSP%20웹%20프로젝트): 회원가입과 로그인 흐름, 세션 처리, 회원 목록 관리가 포함된 JSP 프로젝트입니다.
- [머신러닝 포트폴리오](./머신러닝%20포트폴리오): 분류, 회귀, 신경망, 기초 개념 실습을 R 코드로 정리한 폴더입니다. 알고리즘별로 폴더를 나누어 파일 목적이 한눈에 보이도록 구성했습니다.
- [날씨와 지하철 혼잡도 분석](./날씨와%20지하철%20혼잡도%20분석): Python 노트북으로 진행한 데이터 분석 프로젝트입니다. 지표와 그래프를 통해 결과를 해석할 수 있습니다.
