# 포트폴리오

카카오테크캠퍼스 4기 지원을 위해 정리한 포트폴리오 저장소입니다.  
C 기반 응용 프로그램, JSP 기반 웹 애플리케이션, 머신러닝 모델 실습, 데이터 분석 프로젝트를 한 저장소 안에서 보기 쉽게 정리했습니다.

각 폴더에는 별도의 `README.md`를 두어 프로젝트 목적, 핵심 파일, 실행 화면 또는 결과 예시를 바로 확인할 수 있도록 구성했습니다.

## 대표 프로젝트

### [JSP 기반 웹 애플리케이션](./JSP%20기반%20웹%20애플리케이션)
회원가입, 로그인, 회원 목록 관리, 로그아웃 흐름을 JSP로 구현한 웹 프로젝트입니다.  
페이지 역할이 분명하게 나뉘어 있어 웹 서비스의 기본 구조를 이해하기 쉽고, 화면 예시도 함께 정리해 두었습니다.

### [C 개발 프로젝트](./C%20개발%20프로젝트)
플래시 메모리 시뮬레이션, 전화번호 관리 프로그램, 비디오 대여 관리 시스템처럼 콘솔 환경에서 직접 동작하는 프로그램을 모아 둔 폴더입니다.  
메뉴 구성, 구조체 활용, 파일 입출력, 기능 분리 방식이 드러나는 파일들을 중심으로 정리했습니다.

### [날씨 기반 지하철 혼잡도 분석](./날씨%20기반%20지하철%20혼잡도%20분석)
날씨 정보와 지하철 혼잡도 사이의 관계를 분석한 데이터 프로젝트입니다.  
데이터 정리, 모델 학습, 성능 확인, 결과 해석 흐름을 노트북 하나로 확인할 수 있습니다.

## 전체 구성

- [C 개발 프로젝트](./C%20개발%20프로젝트): C와 C++로 작성한 콘솔 프로그램 모음입니다. 프로그램 흐름 설계와 파일 기반 데이터 관리 방식을 확인할 수 있습니다.
- [자료구조와 알고리즘 구현](./자료구조와%20알고리즘%20구현): 연결 리스트, 큐, 힙, DFS, BFS를 직접 구현한 코드 모음입니다. 자료구조와 탐색 알고리즘의 기본 구현 방식을 확인할 수 있습니다.
- [JSP 기반 웹 애플리케이션](./JSP%20기반%20웹%20애플리케이션): 회원가입과 로그인 흐름, 세션 처리, 회원 목록 관리가 포함된 JSP 프로젝트입니다.
- [머신러닝 포트폴리오](./머신러닝%20포트폴리오): 분류 모델, 회귀 모델, 신경망 모델, 학습 원리와 평가 관련 R 코드를 정리한 폴더입니다.
- [날씨 기반 지하철 혼잡도 분석](./날씨%20기반%20지하철%20혼잡도%20분석): Python 노트북으로 진행한 데이터 분석 프로젝트입니다. 지표와 그래프를 통해 결과를 해석할 수 있습니다.

## 정리 기준

- 각 프로젝트 폴더 안에서 파일 역할과 보는 순서를 바로 파악할 수 있도록 README를 따로 작성했습니다.
- 실행 화면과 결과 이미지는 각 프로젝트 내부의 `화면 예시`, `실행 결과 예시`, `분석 결과 이미지` 폴더로 분리해 의미가 분명하게 보이도록 정리했습니다.
- 빌드 결과물과 개인 메모, 원본 발표 자료처럼 GitHub에 꼭 올릴 필요가 없는 파일은 `.gitignore`로 제외했습니다.
