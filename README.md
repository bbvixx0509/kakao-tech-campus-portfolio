# 포트폴리오

카카오테크캠퍼스 4기 지원을 위해 정리한 포트폴리오 저장소입니다.  
프로젝트와 실습 결과물을 GitHub에서 바로 읽기 쉽게 다시 정리했습니다.

## 대표 프로젝트

### [jsp-web-portfolio](./jsp-web-portfolio)
회원가입, 로그인, 회원 관리 흐름을 구현한 JSP 웹 프로젝트입니다.  
페이지 역할이 분명해서 처음 보는 사람도 구조를 따라가기 쉽고, 웹 서비스의 기본 흐름을 확인하기 좋습니다.

### [c-development-projects](./c-development-projects)
전화번호부, 비디오 대여, 플래시 메모리 실습처럼 직접 동작하는 콘솔 프로그램 모음입니다.  
파일 저장, 메뉴 구성, 기능 분리 같은 기본 구현 역량을 보기 좋게 담고 있습니다.

### [weather-subway-crowding-analysis](./weather-subway-crowding-analysis)
날씨와 지하철 혼잡도의 관계를 살펴본 데이터 분석 프로젝트입니다.  
주제를 정하고 데이터를 정리한 뒤 결과를 해석하는 흐름을 한 번에 볼 수 있습니다.

## 전체 구성

- [machine-learning-portfolio](./machine-learning-portfolio): 분류, 회귀, 신경망, 기초 개념 실습을 R 코드로 정리한 폴더
- [c-algorithm-practice](./c-algorithm-practice): 연결 리스트, 큐, 힙, DFS, BFS를 직접 구현한 실습 코드 모음

## 추천해서 보는 순서

1. `jsp-web-portfolio`
2. `c-development-projects`
3. `weather-subway-crowding-analysis`
4. `machine-learning-portfolio`
5. `c-algorithm-practice`

## 사용 기술

- `C`, `C++`, `Java/JSP`, `Python`, `R`
- `Visual Studio`, `Bootstrap`, `Jupyter Notebook`

## 참고

- 빌드 결과물과 임시 파일은 `.gitignore`로 제외했습니다.
- 일부 코드에는 이해를 돕는 짧은 설명 주석을 추가했습니다.
- 자세한 설명은 각 폴더의 `README.md`에 정리했습니다.
