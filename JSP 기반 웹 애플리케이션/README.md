# JSP 기반 웹 애플리케이션

JSP와 Bootstrap을 활용해 구현한 회원 관리형 웹 프로젝트입니다.  
회원가입부터 로그인, 회원 목록 확인, 삭제, 로그아웃까지 이어지는 기본 흐름을 페이지 단위로 나누어 구성했습니다.

## 프로젝트 개요

이 프로젝트는 웹 애플리케이션의 기본 동작 흐름을 직접 구현해보는 데 초점을 두었습니다.  
화면 페이지와 처리 페이지를 분리하고, 세션을 활용해 로그인 상태를 유지하는 방식으로 구성했습니다.

## 주요 기능

- 회원가입과 아이디 중복 확인
- 로그인과 로그인 결과 처리
- 세션 기반 로그인 상태 유지
- 회원 목록 확인과 회원 삭제
- 로그아웃 처리

## 확인하면 좋은 파일

- `project_intro.jsp`: 프로젝트 소개와 시작 화면 역할을 담당합니다.
- `signup_form.jsp`: 회원가입 입력 화면입니다.
- `check_id_availability.jsp`: 아이디 중복 여부를 확인하는 처리 파일입니다.
- `signup_process.jsp`: 회원가입 입력값을 저장하는 처리 파일입니다.
- `login.jsp`: 로그인 화면입니다.
- `login_process.jsp`: 로그인 요청과 세션 생성을 처리합니다.
- `login_result.jsp`: 로그인 성공 여부와 결과 화면을 보여줍니다.
- `member_dashboard.jsp`: 회원 목록과 관리 기능이 연결되는 메인 화면입니다.
- `delete_member.jsp`: 회원 삭제를 처리합니다.
- `logout.jsp`: 세션 종료와 로그아웃을 담당합니다.
- `department_home.jsp`: 메인 홈 화면입니다.

## 이 프로젝트에서 강조하고 싶은 점

- 기능별로 페이지 역할을 나누어 웹 흐름을 분명하게 구성했습니다.
- 로그인과 회원 관리 흐름이 자연스럽게 이어지도록 구현했습니다.
- 화면과 처리 로직을 분리해 JSP 프로젝트의 기본 구조를 이해하고 적용했습니다.
