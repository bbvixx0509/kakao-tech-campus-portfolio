# 날씨와 지하철 혼잡도 분석

날씨 변수와 지하철 혼잡도 사이의 관계를 분석한 데이터 프로젝트입니다.  
공공 데이터를 활용해 주제를 정하고, 데이터를 정리한 뒤, 모델 성능을 확인하고 결과를 해석하는 과정을 노트북 형태로 정리했습니다.

## 핵심 파일

- `분석 프로그램 코드.ipynb`: 데이터 전처리, 모델 학습, 성능 평가, 시각화가 모두 들어 있는 메인 노트북입니다.

## 프로젝트에서 다룬 내용

- 날씨와 시간대 정보가 지하철 혼잡도 예측에 어떤 영향을 주는지 분석
- 학습 데이터와 검증 데이터를 나누어 모델 성능을 비교
- 예측값과 실제값, 변수 중요도, 상관관계를 그래프로 확인

## 결과 요약

- 검증 기준 `MAE`: `1.7951`
- 검증 기준 `R^2`: `0.9732`
- 검증 기준 `RMSE`: `3.1501`

위 결과는 예측값이 실제 혼잡도와 전반적으로 비슷한 흐름을 보였고, 날씨와 시간 관련 정보가 혼잡도 예측에 의미 있게 작용했음을 보여 줍니다.
