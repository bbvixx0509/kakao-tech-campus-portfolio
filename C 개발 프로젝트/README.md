# C 개발 프로젝트

C와 C++로 작성한 콘솔 프로그램을 모아 둔 폴더입니다.  
단순한 예제 코드보다는 메뉴를 구성하고 데이터를 저장하고 불러오며, 기능별로 파일을 나누어 프로그램 형태로 구현한 결과물을 중심으로 정리했습니다.

## 포함된 프로젝트

- `Flash`: 플래시 메모리의 읽기, 쓰기, 삭제 흐름을 콘솔 메뉴로 다룬 실습 프로젝트입니다.
- `phonebook`: 전화번호 추가, 수정, 삭제, 검색, 정렬 기능을 구현한 전화번호부 프로그램입니다.
- `vedioborrow`: 회원 로그인과 비디오 대여 기능을 파일 기반으로 처리하는 콘솔 프로그램입니다.

## 프로젝트별로 보면 좋은 파일

- `Flash/flash_console.cpp`: 사용자 메뉴를 출력하고 입력을 받아 전체 실행 흐름을 담당하는 파일입니다.
- `Flash/flash_memory.cpp`: 플래시 메모리 읽기, 쓰기, 삭제 동작을 실제로 처리하는 핵심 구현 파일입니다.
- `Flash/header.h`: 플래시 메모리 프로젝트에서 공통으로 사용하는 선언과 구조를 모아 둔 헤더 파일입니다.
- `phonebook/phonebook/phonebook_app.c`: 전화번호부 메뉴, 입력 처리, 검색과 정렬 흐름이 들어 있는 메인 파일입니다.
- `vedioborrow/vedioborrow/video_rental_app.c`: 비디오 대여 프로그램의 메인 메뉴와 사용자 동작을 처리하는 파일입니다.
- `vedioborrow/vedioborrow/video_rental_data.c`: 회원 정보와 비디오 데이터를 읽고 저장하는 파일 처리 로직이 들어 있는 파일입니다.
- `vedioborrow/vedioborrow/video_rental.h`: 비디오 대여 프로젝트에서 사용하는 구조체와 함수 선언을 모아 둔 헤더 파일입니다.

## 이 폴더에서 확인할 수 있는 점

- 구조체를 사용해 여러 항목의 데이터를 관리하는 방식
- 메뉴 선택에 따라 기능을 분리하고 연결하는 콘솔 프로그램 구성 방식
- 텍스트 파일을 이용해 데이터를 저장하고 다시 불러오는 기본적인 파일 입출력 처리 방식

## 실행 예시

### `phonebook`
```text
1. 입력
2. 수정
3. 삭제
4. 추가
5. 이름 오름차순 정렬
6. 검색
7. 출력
```

전화번호부 프로젝트는 메뉴 단위로 기능을 분리해 두어, 콘솔 환경에서도 프로그램 흐름이 비교적 명확하게 보이도록 구성했습니다.

## 사용 환경

- `C`, `C++`
- `Visual Studio`
