# C 개발 프로젝트

C와 C++로 구현한 콘솔 프로그램을 정리한 폴더입니다.  
메뉴 기반 프로그램 설계, 구조체 활용, 파일 저장과 불러오기, 기능 분리 방식이 잘 드러나는 프로젝트를 중심으로 묶었습니다.

## 프로젝트 개요

### 플래시 메모리 시뮬레이션
플래시 메모리의 읽기, 쓰기, 삭제 흐름을 콘솔 환경에서 시뮬레이션한 프로젝트입니다.  
메뉴 입력에 따라 메모리 동작을 직접 확인할 수 있도록 구성했습니다.

### 전화번호 관리 프로그램
전화번호 추가, 수정, 삭제, 검색, 정렬 기능을 제공하는 관리 프로그램입니다.  
구조체 기반 데이터 관리와 메뉴 흐름 설계를 확인할 수 있습니다.

### 비디오 대여 관리 시스템
회원 로그인과 비디오 목록 관리, 대여 처리 흐름을 파일 기반으로 구현한 콘솔 프로그램입니다.  
데이터 저장 구조와 기능별 파일 분리가 분명하게 드러나는 프로젝트입니다.

## 핵심 구현 포인트

- 메뉴 선택에 따라 기능이 자연스럽게 이어지는 콘솔 인터페이스 구성
- 구조체와 배열을 활용한 데이터 관리
- 텍스트 파일을 이용한 저장, 조회, 수정 처리
- 기능별 소스 파일 분리를 통한 프로그램 구조 정리

## 확인하면 좋은 파일

- `플래시 메모리 시뮬레이션/flash_console.cpp`: 사용자 입력과 전체 메뉴 흐름을 담당하는 파일입니다.
- `플래시 메모리 시뮬레이션/flash_memory.cpp`: 읽기, 쓰기, 삭제 동작을 처리하는 핵심 구현 파일입니다.
- `플래시 메모리 시뮬레이션/flash_memory.h`: 공통 구조와 함수 선언을 정리한 헤더 파일입니다.
- `전화번호 관리 프로그램/phonebook/phonebook_app.c`: 전화번호부 기능 전반을 연결하는 메인 구현 파일입니다.
- `비디오 대여 관리 시스템/vedioborrow/video_rental_app.c`: 로그인과 메뉴 흐름을 포함한 메인 파일입니다.
- `비디오 대여 관리 시스템/vedioborrow/video_rental_data.c`: 회원 정보와 비디오 데이터를 저장하고 불러오는 처리 파일입니다.
- `비디오 대여 관리 시스템/vedioborrow/video_rental.h`: 공통 구조체와 함수 선언을 정리한 헤더 파일입니다.

## 이 폴더에서 볼 수 있는 역량

- 콘솔 프로그램 구조를 끝까지 설계하고 구현하는 능력
- 데이터 저장 방식과 메뉴 흐름을 함께 고려하는 문제 해결 방식
- 여러 기능을 하나의 프로그램 안에서 분리해 관리하는 코드 구성 능력

## 사용 환경

- `C`, `C++`
- `Visual Studio`
