# 날씨 기반 지하철 혼잡도 분석

날씨 변화와 지하철 혼잡도 사이의 관계를 분석하고, 혼잡도를 예측하는 모델을 검증한 프로젝트입니다.  
공공 데이터를 정리하고 예측 결과를 수치와 시각화로 확인할 수 있도록 Jupyter Notebook 형태로 구성했습니다.

## 프로젝트 개요

이 프로젝트는 날씨 정보가 지하철 이용량과 혼잡도에 어떤 영향을 주는지 확인하는 데서 출발했습니다.  
전처리, 학습, 검증, 시각화까지 한 흐름으로 연결해 분석 과정을 정리했습니다.

## 핵심 내용

- 날씨 데이터와 혼잡도 데이터를 결합해 분석용 데이터셋 구성
- 학습 데이터와 검증 데이터를 나누어 예측 성능 확인
- 예측값과 실제값 비교를 통한 모델 신뢰도 검토
- 변수 중요도와 상관관계 시각화를 통한 결과 해석

## 핵심 파일

- `분석 프로그램 코드.ipynb`: 데이터 전처리, 모델 학습, 검증, 시각화가 모두 포함된 메인 노트북입니다.

## 결과 요약

- 검증 기준 `MAE`: `1.7951`
- 검증 기준 `R^2`: `0.9732`
- 검증 기준 `RMSE`: `3.1501`

위 결과를 통해 예측값이 실제 혼잡도 흐름을 비교적 안정적으로 따라가고 있음을 확인했습니다.  
특히 날씨와 시간 관련 정보가 혼잡도 예측에 유의미하게 반영된다는 점을 시각화 결과로 함께 확인할 수 있습니다.
