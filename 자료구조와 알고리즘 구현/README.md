# 자료구조와 알고리즘 구현

연결 리스트, 큐, 힙, DFS, BFS를 직접 구현한 C 코드 모음입니다.  
기본 개념을 정리하는 수준에 그치지 않고, 자료구조 동작과 탐색 흐름을 코드로 끝까지 구현한 결과를 담고 있습니다.

## 구성 내용

- `score_linked_list.c`, `score_linked_list.h`: 성적 데이터를 연결 리스트로 관리하는 프로그램입니다.
- `linked_queue.c`: 연결 리스트 기반 큐 구현 파일입니다.
- `max_heap.c`: 최대 힙 생성과 삽입, 삭제를 다루는 파일입니다.
- `DFS,BFS/bfs`: 너비 우선 탐색 구현 프로젝트입니다.
- `DFS,BFS/dfs`: 깊이 우선 탐색 구현 프로젝트입니다.

## 확인하면 좋은 포인트

- 포인터와 구조체를 이용해 자료구조를 직접 구성한 방식
- 탐색 순서와 자료 저장 방식이 코드에서 어떻게 표현되는지
- 기본 개념을 실제 동작 가능한 프로그램 형태로 구현한 과정

## 이 폴더의 의미

대표 프로젝트처럼 화면 중심으로 보이는 폴더는 아니지만, 구현 기초와 문제 해결 방식을 확인할 수 있는 코드 포트폴리오입니다.  
특히 이후의 콘솔 프로그램과 데이터 처리 프로젝트를 이해하는 바탕이 되는 구현 역량을 보여줍니다.
