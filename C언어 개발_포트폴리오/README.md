# C언어 개발 포트폴리오

콘솔 환경에서 직접 구현한 C/C++ 프로젝트를 모아둔 폴더입니다.  
구조체, 배열, 파일 입출력, 메뉴형 인터페이스 구현 경험을 확인할 수 있습니다.

## Projects

| 프로젝트 | 설명 | 주요 파일 |
| --- | --- | --- |
| `Flash` | 플래시 메모리의 초기화, 쓰기, 읽기, 삭제를 실습 형태로 구현한 프로젝트 | `Flash/main.cpp`, `Flash/header.cpp`, `Flash/header.h` |
| `phonebook` | 전화번호 저장, 수정, 삭제, 검색, 정렬 기능을 포함한 콘솔 전화번호부 | `phonebook/phonebook/phonebook.c` |
| `vedioborrow` | 회원가입, 로그인, 비디오 대여/반납, 목록 조회 기능이 있는 프로그램 | `vedioborrow/vedioborrow/소스.c`, `vedioborrow/vedioborrow/header.c`, `vedioborrow/vedioborrow/header.h` |

## What To Look At

- 파일 저장을 활용한 데이터 유지 방식
- 메뉴 기반 프로그램 흐름 구성
- 구조체와 배열을 이용한 데이터 관리
- 기능 단위 함수 분리

## Environment

- Visual Studio
- C / C++
