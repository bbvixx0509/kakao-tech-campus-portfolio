# C Development Projects

C와 C++로 작성한 콘솔 프로그램을 모아둔 폴더입니다.  
기능을 직접 구현하면서 구조체, 배열, 파일 저장, 메뉴형 인터페이스를 연습한 결과물을 담았습니다.

## 들어 있는 프로젝트

- `Flash`: 플래시 메모리 동작을 간단하게 실습한 프로젝트입니다. 초기화, 쓰기, 읽기, 삭제 흐름을 확인할 수 있습니다.
- `phonebook`: 전화번호를 저장하고 수정, 삭제, 검색, 정렬까지 할 수 있는 콘솔 전화번호부입니다.
- `vedioborrow`: 회원가입, 로그인, 비디오 대여와 반납 기능을 넣은 콘솔 프로그램입니다.

## 보면 좋은 부분

- 데이터를 파일로 저장하고 다시 불러오는 방식
- 메뉴 선택에 따라 기능이 이어지는 콘솔 흐름
- 구조체와 함수 분리로 기능을 나눈 방식
- 작은 프로그램이라도 끝까지 동작하게 만들려는 구현 습관
