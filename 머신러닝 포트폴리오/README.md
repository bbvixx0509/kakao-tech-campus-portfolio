# 머신러닝 포트폴리오

R로 구현한 머신러닝 코드와 모델 비교 결과를 주제별로 정리한 폴더입니다.  
분류, 회귀, 신경망, 모델 성능 비교 관련 코드를 한눈에 보기 쉽도록 구조화해 두었습니다.

## 구성 방향

이 폴더는 단순한 코드 모음이 아니라, 서로 다른 모델을 어떤 기준으로 비교하고 어떤 결과를 확인했는지 보여주기 위한 포트폴리오 형태로 정리했습니다.  
비슷한 성격의 코드를 묶어서 읽는 사람이 모델 유형별 흐름을 빠르게 이해할 수 있도록 구성했습니다.

## 폴더 안내

### `분류 모델`

- `knn_distance_comparison.R`: k-NN에서 거리 계산 방식에 따라 분류 결과가 어떻게 달라지는지 비교합니다.
- `logistic_vs_knn.R`: 로지스틱 회귀와 k-NN의 분류 결과를 비교합니다.
- `kernel_svm.R`: 커널 SVM을 적용해 비선형 분류 문제를 다룹니다.

### `회귀 모델`

- `linear_vs_tree.R`: 선형 회귀와 회귀 트리를 비교합니다.
- `random_forest.R`: 랜덤 포레스트 회귀 모델의 예측 성능을 확인합니다.

### `신경망 모델`

- `backpropagation.R`: 역전파 계산 흐름을 단계별로 확인합니다.
- `neural_network_forward.R`: 순전파 계산 과정을 직접 확인합니다.
- `neural_network_regression.R`: 신경망 기반 회귀 예측을 수행합니다.

### `모델 원리와 성능 평가`

- `margin_perceptron.R`: 마진 퍼셉트론의 분류 경계를 확인합니다.
- `perceptron_learning.R`: 퍼셉트론 학습 과정을 단계별로 점검합니다.
- `information_gain.R`: 정보 이득 계산을 통해 분할 기준을 비교합니다.

## 이 폴더에서 강조하고 싶은 점

- 서로 다른 모델을 같은 문제에 적용하며 결과를 비교한 경험
- 정확도, 손실값, 오차 지표 등 기본 성능 지표를 직접 확인한 과정
- 모델별 계산 흐름을 코드 수준에서 이해하고 정리한 구성 방식

## 참고

- 일부 코드에는 `caret`, `class`, `kknn`, `nnet`, `e1071`, `rpart`, `randomForest`, `FSelector` 패키지가 사용됩니다.
- 데이터 파일이 필요한 경우에는 별도의 `.RData` 파일을 불러오는 형태로 작성된 스크립트가 있습니다.
